# Support

Use the smallest channel that fits the problem.

## Bugs

Open a [GitHub issue](https://github.com/your-org/MUTX/issues/new/choose) with the bug template.

Please include:

- what you expected
- what happened instead
- repro steps
- environment details

## Feature Requests

Open a [GitHub issue](https://github.com/your-org/MUTX/issues/new/choose) with the feature template.

Tie proposals to a real user or maintainer problem when possible.

## Docs Improvements

Open a docs issue or send a PR directly to the `docs/` folder in the repository.
