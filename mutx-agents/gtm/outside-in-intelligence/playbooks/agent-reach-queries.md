# Agent Reach Query Pack

Use these as starting points, not as mandatory loops.

## X / Twitter
- `bird search 'agent runtime OR agent control plane OR agent observability OR ai governance' -n 10`
- `bird search 'human in the loop agents OR HITL agents OR agent approvals' -n 10`
- `bird search 'production agents OR long-running agents OR agent retries OR agent failures' -n 10`
- `bird search 'agent orchestration accountability OR audit trail agents' -n 10`

## Reddit
- Search via Exa / mcporter for:
  - production agents reliability
  - AI agents in production failures
  - human in the loop agents
  - agent orchestration debugging
  - LLM agent governance

## GitHub
- `gh search repos 'agent runtime' --limit 10`
- `gh search issues 'agent observability' --limit 10`
- `gh search issues 'human in the loop agents' --limit 10`
- `gh search issues 'agent retries' --limit 10`

## YouTube / video
- look for fresh transcripts/tutorials on:
  - agent orchestration
  - long-running agents
  - human approval flows
  - agent observability
  - agent governance

## Web / RSS
Look for:
- vendor blog posts on agent ops
- engineering incident writeups
- posts where teams explain why agent demos broke in real workflows

## Output filter
Only keep signals that help MUTX answer one of these:
1. What pain is real and recurring?
2. What language do buyers/operators already use?
3. What thread is worth replying to?
4. What product/runtime truth does the market keep demanding?
5. What account or design-partner trigger just appeared?
