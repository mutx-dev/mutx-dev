# ClawHub Skills — Install Guide

Run these via `clawhub install <skill-name>` in sequence.

## Phase 1 — Core Skills (Install Now)
```
clawhub install lead-generation
clawhub install agent-browser-core
```

## Phase 2 — Visibility Sprint (Install Before AI Audit)
```
clawhub install seo-research-master
```

## Phase 3 — Social Setup (Install Before Priority 3)
```
clawhub install ecommerce-social-media-marketing
```

## Optional — If Sheri Provides Amazon Access
```
clawhub install amazon-advertising-strategy
```

---

## Information Needed Before Each Phase

### Before Starting Priority 1 (B2B Lead Gen)
- Sheri's LinkedIn profile URL (for outreach template tone)
- Any current prospect list she already has (to avoid duplicates)
- Industry focus: domestic US, international, or both?

### Before Starting Priority 2 (AI Visibility Audit)
- Earthborn Products website URL
- List of competitors she knows about (she mentioned Sovereign Silver)

### Before Starting Priority 3 (Social Setup)
- Does she have existing business pages on FB/IG/X or are these new accounts?
- What is her current posting frequency (if any)?
- Who is her target audience per platform?

---

## Agent Instructions for Skill Usage

**lead-generation**: Use to research and identify B2B white-label supplement manufacturers. Do NOT run live campaigns — research and draft only. Output: prospect list + outreach templates.

**agent-browser-core**: Use for AI visibility audit — open browser, query AI models, document which competitors appear and why.

**seo-research-master**: Use to identify content gaps on Earthborn Products website after audit.

**ecommerce-social-media-marketing**: Use to generate a 30-day content calendar for Facebook, Instagram, and X (Twitter) for B2B supplement audience.

**amazon-advertising-strategy**: Only if Sheri provides her Amazon Ads account credentials or access.
