### Datetime Timezone Mismatch
- **Error**: `can't subtract offset-naive and offset-aware datetimes`
- **Root cause**: `created_at` column used `DateTime(timezone=True)` but `updated_at` used plain `DateTime` (timezone-naive)
- Both columns were being passed timezone-aware datetimes, causing asyncpg to fail
- **Fix**: Added `DateTime(timezone=True)` to all `updated_at` columns in models.py (commit 1b439d97)

### JSON Serialization Error
- **Error**: `TypeError: Object of type datetime is not JSON serializable`
- **Root cause**: Exception handler was returning datetime objects in JSON response
- **Fix**: Changed `datetime.now(timezone.utc)` to `datetime.now(timezone.utc).isoformat()` in exception_handlers.py (commit 1b439d97)
